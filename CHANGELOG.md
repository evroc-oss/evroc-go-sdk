# Changelog

## [0.1.0] - 2026-01-13

Initial release of evroc Go SDK

- Compute API (VirtualMachines, Disks, PlacementGroups, HotswapDiskAttachments)
- Networking API (VirtualPrivateClouds, Subnets, SecurityGroups, PublicIPs)
- IAM API (PermissionSets, Projects)
- Storage API (Buckets, BucketServiceAccounts)
- OAuth2/OIDC authentication with token refresh
- Configuration via environment variables or YAML file
